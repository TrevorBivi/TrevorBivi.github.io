What it is
 This is a first-person shooter game created with the Unity 3D engine. Players shoot an endless hoard of zombies that can run, climb and jump to get to them. I first created a single player version then re-wrote most of it to work with multiplayer. Here are some of the features Iâ€™ve created

  • Helicopter AI that shoots visible enemies



Sample gameplay

How it works
 This project made extensive use of object oriented programming and recursive functions to develop the enemies. A recursive node based pathfinding algorithm close to Dijkstra's algorithm is used for determining the best paths in between all nodes so enemy objects can reach players anywhere. Furthermore, recursive functions are used to determine where enemies were hit when they are shot and to spawn a ragdoll object with bones in the right position when enemies die.
