Tetris Bot
Languages used
 Python extended with C.

What it is
 This is an educational Super Mario style phone app I lead the creation of as part of a group of three.


The game. An unlock all levels option and keyboard controls have been added.
Move left: A or left arrow
Move right: D or right arrow
Jump: W, up arrow or space bar
How it works
 This is another example of object oriented programming. I was responsible for the creation of a collision engine, special interactive objects like springs and moving platforms as well as a cookie based save system to remember the unlocked levels.
